History

A Begining...
Everything started deep in the internet, somewhere close to date 26 Feb. 2019 by guy named Polivilas posted his work on cheating forum where he posted a link to his github. This is where everything started. -

First steps...
People were using the code over a 6-8 months, then a EmuTarkov community has risen. TheMaoci created Discord community and InNoHurryToCode created Github Organisation. We all Agreed that JustEmuTarkov will be a good name for it. So we started a work.

Offline but with Progression
Oh man then i come back to this time it was a good time... Reversing game to be able to save progress. First attempts was preety archaic but worked :) (Video #1, Video #2, Video #3) All thanks for this ideas to TheMaoci and BALIST0N

Fixing Fixing Fixing...
It was never ending story of fixing. We fix one issue next one pops up or even 2... (actually i (TheMaoci) go full on rampage mode with fixing, spending over 80% of my time for coding, fixing and helping others)

The Rising
Well Community has risen quite fast, we even didnt noticed when we have over 8000+ people on our discord.

The Fall
Dev's of JET started to disagree with what should be shared, implemented and what shouldnt, so community shatter in few peaces (I left the project completly at this time(with closing discord), watching it from a shadow what was happening ~TheMaoci). Well that was the worst possible thing to do... split community. InNoHurryToCode started full open source on JustEmuTarkov Github Organisation sharing reversed data (which is against github TOS and propably most of the services). It ended with DMCA TakeDown.

A Black hole
... ... ... ...

Comeback
Well by haveing alot of people reaching me out to bring back the old community back. I ended up by creating it again and bringing back all people who are willing to help with the project with restrictions to be that the project will stay in private section (to not get dmca takedowns like the ones previously done by idiotic behaviour). Everyone who want to contribute to the project will get access to it just by talking to TheMaoci. About Repositories Server has beed rewrited slightly to be easly editable + still has low amount of files(i will never fully restructure the server unless it will be required cause of some issues). We are also having a properly code launcher (which actually tells user what is wrong and why he cannot start a game, etc. Also we are working on Multiplayer aspect of the emulator (it will be very private after we finish it fully) - what i mean by "very private" ? well you will be able to get it from one of creators or from people who already got it.

History section still needs some polishing and is still not finished.
